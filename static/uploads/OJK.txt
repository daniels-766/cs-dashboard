Batas Waktu :

1. Pelapor wajib menyampaikan laporan debitur secara bulanan paling lambat tanggal 12 bulan berikutnya setelah bulan laporan debitur
2. Koreksi jika ada temuan paling lambat tanggal 12 di bulan berikutnya
3. Jika Waktu laporannya hari libur, maka akan ke hari kerja berikutnya
4. Disiapkan/dilapor lebih awal biar mencegar antrian

1. Di web SLIK
2. Jika ada error, download file error lalu di perbaiki di aplikasi lalu buat lagi file txt lalu diupload ulang di SLIK Reporting
